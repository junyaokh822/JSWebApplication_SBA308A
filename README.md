# Dog Breed Finder

A modern, interactive web application that allows users to search and explore different dog breeds using The Dog API.

## Features

- **Search Functionality**: Search for dog breeds by name
- **Image Carousel**: Browse through multiple high-quality images for each breed (up to 10 images)
- **Breed Information**: View detailed information including lifespan, temperament, origin, breed group, and description
- **Interactive UI**:
  - Click arrow buttons to navigate between images
  - Click thumbnails to jump to specific images
  - Smooth fade transitions between images
- **Responsive Design**: Works seamlessly on desktop

## Technologies Used

- **HTML5**: Structure and semantic markup
- **CSS3**: Modern styling with gradients, animations, and responsive design
- **JavaScript (ES6 Modules)**: Modular code organization
- **Axios**: HTTP client for API requests
- **The Dog API**: External API for dog breed data and images

## Project Structure

## How to Run

1. Clone or download this repository
2. Open `index.html` in a modern web browser
3. Start searching for your favorite dog breeds!

## Design Highlights

- Beautiful purple-to-pink gradient background
- Smooth animations and hover effects
- Card-based layout with glassmorphism effects

## API

This project uses [The Dog API](https://thedogapi.com/) to fetch breed information and images.
